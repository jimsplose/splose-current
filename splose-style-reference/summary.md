## Usage Tips for Claude

When recreating Splose UI in a React prototype:

1. Start with **css-variables.md** to set up your CSS custom properties
2. Reference **colours.md** and **typography.md** for exact token values
3. Check **components/{type}.md** for the computed styles of specific elements
4. Use **screenshots/** as visual targets to match against
5. Use **page-structures/** to understand layout hierarchy
