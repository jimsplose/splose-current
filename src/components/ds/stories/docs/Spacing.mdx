## Component Spacing Conventions

| Context | Spacing | Class |
|---|---|---|
| Page padding | 24px | `p-6` |
| Card padding (default) | 16px | `p-4` |
| Form field gap | 16px | `gap-4` / `space-y-4` |
| Button internal | 16px horizontal, 8px vertical | `px-4 py-2` |
| Table cell | 16px horizontal, 12px vertical | `px-4 py-3` |
| Section gap | 24px | `gap-6` / `mb-6` |
| Sidebar width | 256px | `w-64` |
| Header height | 48px | `h-12` |
