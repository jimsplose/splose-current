# Typography

Splose uses two font families and a semantic naming system for typography styles.

## Font Families

| Family | Font | Usage | Tailwind |
|---|---|---|---|
| **Display** | Sprig Sans (Bold) | Page titles, hero text | `font-display` |
| **Sans** | Inter | Everything else | `font-sans` |
